Secure Password Generator

The Secure Password Generator is a web application designed to create strong, random passwords tailored to user specifications. Users can specify the desired password length and choose to include various character types, including uppercase letters, lowercase letters, numbers, and symbols. The application ensures that generated passwords are secure and difficult to crack, promoting better security practices. If the user selects a password length of less than 16 characters, an alert will notify them to choose a more secure length. The interface is user-friendly, providing real-time feedback and easy password copying functionality.

Technologies Used: HTML, CSS, JavaScript
